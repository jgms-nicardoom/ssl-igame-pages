# ssl-igame-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定网站或域名，仅作为静态页面资源进行存储与分发。

## 项目简介

`ssl-igame-pages` 是一个纯静态页面集合仓库。所有页面均为独立 HTML 文件，适合用于学习、测试、参考或轻量级前端展示。仓库不包含服务器端逻辑或数据库依赖，仅通过 GitHub Pages 等静态托管服务即可直接访问。

## 目录说明

```
├── index.html          # 仓库首页/导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：可选的仓库入口页面，用于展示页面列表或导航。
- `pages/`：所有独立 HTML 页面存放于此，每个文件代表一个完整的页面。
- `assets/`：页面共享的样式、脚本及图片资源，按类型分别存放。

## 页面归档说明

- 所有页面均为自包含的 HTML 文件，可直接在浏览器中打开。
- 页面之间无强依赖关系，可单独使用或部署。
- 部分页面可能引用 `assets/` 下的公共资源，请确保路径正确。
- 归档内容以静态展示为主，不涉及动态数据交互。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 开启静态站点后，通过对应 URL 访问。
